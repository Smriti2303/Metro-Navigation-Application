## Overview
The **Metro Navigation Application** is designed to improve the metro travel experience. Unlike traditional metro apps, this app not only provides station navigation but also helps users discover nearby **tourist attractions**, calculates **metro fares**, and offers an **interactive, user-friendly interface** with creative backgrounds to enhance the user experience. Built using **Python** and **Streamlit**, this app provides a seamless, engaging, and functional solution for commuters and tourists alike.

## Features
- **Station Search**: Search for metro stations and view nearby tourist attractions.
- **Fare Calculation**: Calculates the metro fare based on the selected route.
- **Interactive UI**: Engaging and dynamic user interface with creative backgrounds for better interaction.
- **User-Friendly**: Designed to be easy to navigate for both locals and tourists.

## Technologies
- **Programming Language**: Python
- **Library**: Streamlit
- **IDE**: PyCharm

## Installation & Setup

### Prerequisites
- Python 3.x
- Streamlit library

### Steps to Run the App
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/metro-navigation-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd metro-navigation-app
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   streamlit run app.py
   ```
